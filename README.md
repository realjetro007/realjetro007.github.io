# ASCII Art Generator
Turn image file (PNG, JPEG) into ASCII Artworks with controllable brightness, contrast, dithering and the option to invert colours.
This is a short project I have created before I begin study at university.

## Summary
This is a full-stack python and javascript project showcasing my learning in image processing, realtime ui updates and web development and deployment.

## Features:
* Upload any image for ASCII conversion in realtime
* Adjust width, height, brightness and contrast
* Optional dithering which occasionally provides more accurate results

## Tech Stack
* Backend - Python, Flask, Pillow for image processing
* Frontend - HTML, CSS, Javascript
